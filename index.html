# Manifesto

We build tools that dissolve into the workflow. Absolute technical precision masked as simplicity. This is the Browser-Core modernism approach, where every pixel is accountable.

# Installation

npm install @purple-fabric/core
Is this compatible with modern frameworks?

Yes, it exposes headless hooks ensuring 100% compatibility with React, Vue, and Svelte.

How does the synthetic shadow DOM work?

It isolates styles at the AST level before rendering, guaranteeing zero side-effects on the global scope.

What is the bundle footprint?

Core runtime is strictly under 4.2kb gzipped, prioritizing critical path metrics.
