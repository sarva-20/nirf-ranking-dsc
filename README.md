# NIRF Ranking Intelligence - Terminal Workspace (v2026.0)

A high-density, "Browser-Core Modernism" analytical workspace for institutional ranking monitoring. This project transitions traditional NIRF data into a professional, DevTools-inspired technical environment.

![Dashboard Preview](https://img.shields.io/badge/Status-v2026.0--Released-06B6D4?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Built--With-Node.js%20%7C%20Express%20%7C%20Chart.js-white?style=for-the-badge)

## ⚡ Key Features

- **Browser-Core UI:** A sophisticated terminal-style workspace featuring macOS-style traffic lights, a custom address bar, and a 12-column high-density grid.
- **2026 Analytical Cycle:** Full repository support for NIRF 2026 parameters across Engineering and Medical categories.
- **Institutional Inspector:** Real-time property inspection of TLR, RP, GO, OI, and PR scores.
- **NIRF AI Analysis Engine:** A professional-grade AI validation tool powered by Purple Fabric, featuring a streaming technical console for institutional insights.
- **Dynamic Visualisations:** Synchronised Radar and Bar charts built with Chart.js, optimised for institutional comparison.

## 🛠 Tech Stack

- **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3 (Custom Properties).
- **Backend:** Node.js, Express.
- **Data:** JSON-backed local repository (`data.json`).
- **Visuals:** Chart.js with JetBrains Mono typography integration.

## 🚀 Quick Start

### 1. Installation
Ensure you have Node.js installed, then clone and install dependencies:
```bash
npm install express node-fetch@2
```

### 2. Launch
Start the local intelligence server:
```bash
node server.js
```

### 3. Access
Open your browser and navigate to:
`http://localhost:3000`

## 📁 Project Structure

- `server.js`: Node.js Express server with built-in AI streaming proxy.
- `calculator.js`: Weightage-based ranking logic (30% TLR, 30% RP, etc.).
- `data.json`: The core Institutional Repository (2026 Edition).
- `script.js`: Frontend orchestrator for charts, pagination, and AI logic.
- `styles.css`: The 'Browser-Core' design system.
